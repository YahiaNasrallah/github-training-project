# School Manager App

## Project Description

School Manager App is a simple Flutter application developed to help manage students and classes in a school environment.

The application allows users to add students, organize classes, manage data dynamically, and view an overview dashboard containing useful statistics about students and classes.

The project was built using Flutter and Dart with a clean and user-friendly interface.

---

## Features

* Add new students
* Delete students
* Add and remove classes
* Dynamic class selection
* Overview dashboard with statistics
* Display total number of students and classes
* Bottom Navigation Bar for easy navigation
* Responsive and organized UI
* Multiple screens navigation

---

## Screens Included

### Home Screen

* Add students
* Select classes
* Display recent students
* Delete student records

### Classes Screen

* Add new classes
* Remove classes
* Display all available classes

### Overview Screen

* Show total students count
* Show total classes count
* Display classes with related student numbers
